15. 
    
    𝗮. Let u = x², then u² - 5u + 4 = (u - 1)(u - 4) = (x² - 1)(x² - 4) = (x + 1)(x - 1)(x + 2)(x - 2)
    
    𝗯. Let u = (x - 3), then u² + 6u + 8 = (u + 2)(u + 4) = (x - 3 + 2)(x - 3 + 4) = (x - 1)(x + 1)
    
    𝗰. Let u = (y + 1), then 3u² - 7u + 2 = (3u - 2)(u - 1) = (3(y + 1) - 2)((y + 1) - 1) = (3y + 1)(y)
