## Answer Key

1. a) (x + 4)(x - 4)
   b) (5y + 3)(5y - 3)
   c) (7a + 8b)(7a - 8b)
   d) (11 + 2z)(11 - 2z)

4. a) (x + 3)(x² - 3x + 9)
   b) (2y - 1)(4y² + 2y + 1)
   c) (5a + 2)(25a² - 10a + 4)
   d) (3 - b)(9 + 3b + b²)

7. a) (x + 2)(y + 3)
   b) (a + 3)(b - 5)
   c) (2x + 5)(y - 3)

9. a) (x² + 4)(x + 2)(x - 2)
   b) 2x(x² + 8)
   c) (3y + 5z)(3y - 5z)

11. a) (x³ + y³)(x³ - y³) = (x + y)(x² - xy + y²)(x - y)(x² + xy + y²)
    b) (a³ + b³)(1 - c³) = (a + b)(a² - ab + b²)(1 - c)(1 + c + c²)
    c) (x² + 9y²)(x + 3y)(x - 3y)
    d) 4xy

13. a) (2x + 1)(x + 3)
    b) (3x - 4)(x - 2)
    c) (3x - 2)(2x + 3)
    d) (2x - 3)²

15. a) Let u = x², then u² - 5u + 4 = (u - 1)(u - 4) = (x² - 1)(x² - 4) = (x + 1)(x - 1)(x + 2)(x - 2)
    b) Let u = (x - 3), then u² + 6u + 8 = (u + 2)(u + 4) = (x - 3 + 2)(x - 3 + 4) = (x - 1)(x + 1)
    c) Let u = (y + 1), then 3u² - 7u + 2 = (3u - 2)(u - 1) = (3(y + 1) - 2)((y + 1) - 1) = (3y + 1)(y)
